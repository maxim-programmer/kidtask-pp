# ТЗ на дизайн сайта KidTask

Веб-приложение с тремя зонами: 
- публичная (лендинг, вход, регистрация);
- кабинет родителя;
- кабинет ребёнка. 
Два отдельных визуальных стиля для детского кабинета в зависимости от возрастной группы.

## Публичная зона

Страница входа для родителя:
- логотип kidtask;
- форма: email, пароль;
- кнопка "войти";
- ссылка "нет аккаунта? зарегистрироваться".

Страница регистрации родителя:
- форма: имя, email, пароль;
- кнопка "зарегистрироваться";
- ссылка "уже есть аккаунт? войти".

Страница входа для ребёнка:
- отдельная страница, отличается визуально от родительской;
- форма: логин, пароль;
- крупные поля и кнопка.

Цветовая схема статусов (единая для всех зон):
- активное - синий;
- на проверке - жёлтый;
- требует доработки - оранжевый;
- выполнено - зелёный;
- недостаточно монет - серый.

## Кабинет родителя

Общий стиль:
- чистый, минималистичный;
- цветовая схема: белый фон, акцентный цвет на выбор дизайнера;
- шрифт читаемый, размер комфортный для взрослого.

Дашборд:
- список детей — карточки с именем, балансом, прогрессом;
- кнопка "добавить ребёнка";
- быстрые счётчики задач на проверке, новых целей в вишлисте.

Страница ребёнка:
- имя, баланс монет;
- вкладки: задания / вишлист.

Страница заданий:
- кнопка "создать задание";
- список заданий с фильтром по статусу;
- карточка задания: название, описание, награда, статус, кнопки действий;
- статусы цветами.

Создание задания:
- модальное окно или отдельная страница;
- поля: выбор ребёнка, название, описание, награда в монетах;
- кнопки "создать" / "отмена".

Страница вишлиста:
- список целей ребёнка;
- карточка: название, описание, поле для ввода цены, кнопка "сохранить цену";
- статус "куплено" — кнопка "доставлено".

## Кабинет ребёнка — Младшая группа (7-10 лет)

Общий стиль:
- яркие насыщенные цвета;
- мультяшные иконки и иллюстрации;
- очень крупные кнопки с закруглёнными углами;
- минимум текста, максимум визуала;
- анимации при получении монет (монетки падают, звёздочки);
- игровой шрифт, крупный размер.

Шапка:
- имя ребёнка и аватар;
- баланс монет крупно с иконкой монетки;
- нижняя навигация: задания/мои цели (две большие вкладки с иконками).

Страница заданий:
- карточки заданий крупные, с иконкой;
- название задания крупно;
- награда — иконка монетки + число;
- кнопка озвучки задания 🔊большая кнопка "сделал!" зелёного цвета;
- статус "требует доработки" — оранжевая карточка с комментарием родителя.

Страница вишлиста:
- карточки целей с картинкой;
- название крупно;
- шкала прогресса — визуальная, например монетки заполняются;
- кнопка "хочу это!" при добавлении;
- кнопка "забрать!" когда хватает монет — яркая, анимированная.

## Кабинет ребёнка — Старшая группа (11-14 лет)

Общий стиль:
- современный, в стиле мобильного приложения;
- тёмная или полутёмная тема на выбор дизайнера;
- сдержанные акцентные цвета;
- нормальный размер элементов;
- минимум анимаций, только функциональные;
- современный шрифт.

Шапка:
- имя и баланс;
- навигация: задания/вишлист/прогресс.

Страница заданий:
- список с фильтром по статусу;
- карточка: название, описание, награда, статус;
- кнопка "выполнено";
- комментарий родителя при отклонении.

Страница вишлиста:
- карточки целей;
- прогресс бар к каждой цели;
- баланс монет;
- кнопка "купить" когда хватает монет;
- кнопка "добавить цель".

## Дополнения и уточнения к ТЗ

Системные процессы:
- транзакция: начисление валюты после подтверждения выполнения;
- валидация при входе/регистрации и денежных операциях; 
- хэширование паролей.

Заметки:
- родитель пользуется телефоном (portrait), ребёнок -- телефоном и планшетом (portrait, landscape);
- у ребёнка есть дата рождения, от неё зависит выбор интерфейса ЛК.
